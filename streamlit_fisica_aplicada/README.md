# Simulador de Física Aplicada con Streamlit

Incluye dos temas:

1. Ecuaciones generales del movimiento
2. Ley de Gravitación Universal

## Cómo ejecutarlo

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Qué hace

- Calcula posición final, velocidad final y trayectoria.
- Calcula fuerza gravitatoria, aceleración y velocidad orbital.
- Simula una órbita simple en 2D.
- Incluye procedimiento paso a paso para documentar el trabajo.
